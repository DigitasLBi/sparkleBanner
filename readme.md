Gulp boilerplate
====================

First run:

	npm install

Then: ('development' is the default task)

	gulp

To generata a dist

	gulp prod

To generate icon files, copy SVG files to:

	app\icons


Vendors is placed in:

app\vendor

and included in the page like this:

	<script src="vendors/jquery-1.12.3.min.js"></script>
